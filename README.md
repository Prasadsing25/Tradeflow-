# Tradeflow-
The Tradeflow project is a real-time stock 
trading and portfolio management platform implemented 
using the MERN stack. It features a decoupled architecture 
with a React.js frontend, Node.js/Express.js backend, and 
MongoDB database. Key features include live data 
visualization using Chart.js, secure JWT authentication, 
and full portfolio tracking (Watchlist, Holdings, Positions). 
The system architecture utilizes a decoupled approach, 
featuring a dynamic React.js frontend for client interactions 
and data visualization using Chart.js to plot live and 
historical stock data. The Express.js/Node.js backend 
manages secure user authentication (JWT), handles CRUD 
operations for portfolio management (Watchlist, Holdings, 
Positions), and interfaces with external financial APIs for 
live market data. Data persistence is handled by MongoDB, 
which is integrated via Express to manage user schemas and 
models. 


• Frontend: React.js handles UI and Chart.js 
manages visualization [cite: 360]. 
• Backend: Node.js/Express.js manages backend 
logic and APIs 
• Database: MongoDB stores user and 
transaction orders
